# CharPalette

> Win + . is boring af, so I made this. You're welcome.

## What is CharPalette?

CharPalette is a spicy little Python tool that absolutely demolishes the default Windows emoji picker (yeah, that `Win + .`). Tired of hunting for that one weird character to flex in your code, chat, or docs? CharPalette is here to rescue your Unicode game.

## Features

- **Fast & Furious**: Launch instantly, grab the character you need, get on with your life.
- **Curated Palettes**: All the symbols, emojis, and obscure glyphs you never knew you needed.
- **Super Search**: Fuzzy searching so good, you'll find what you want even if you can't spell it.
- **Copy With One Click**: Click, copy, done. No more awkward copy-paste rituals.
- **Customization**: Want your own palette? Go wild. Make it weird. Make it yours.
- **Python Powered**: Because everything cool should run on Python.

## Why CharPalette?
